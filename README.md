## **Project Documentation: Task Management API**

### **Project Structure**

The project follows a MVC Architecture. Below is the file structure:

```
project/
├── app.js
├── routes/
│   └── taskRoutes.js
├── controllers/
│   └── taskController.js
├── middlewares/
│   └── errorMiddleware.js
└── models/
    └── taskModel.js

```
